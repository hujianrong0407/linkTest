@ResponseBody ---注解在 Spring 框架中用于将控制器方法的返回值直接写入 HTTP 响应体中，而不是被解析为视图名称。这意味着方法的返回值会被转换为 JSON、XML 或其他格式，并作为响应返回给客户端。这通常用于构建 RESTful API。

@Controller 注解---是 Spring 框架中用于标识一个类作为控制器组件的注解。它的主要作用是将一个类标记为 Spring MVC 的控制器，使得该类能够处理 HTTP 请求。被 @Controller 注解标记的类通常用于接收和处理来自客户端的请求，并返回相应的视图或数据。

@Service 是 Springboot1 框架中的一个注解，用于标记服务层的组件。它表明一个类是服务层的 Bean，并由 Spring 容器管理。这个注解通常用于标识业务逻辑组件，使其可以被自动发现和注入到其他类中。
